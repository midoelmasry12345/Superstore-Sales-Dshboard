Sales Performance Dashboard (Power BI)

This project presents a Power BI dashboard analyzing sales performance. It includes KPIs such as Total Sales, Total Profit, Quantity Sold, and product rankings. An actionable insight message is added to automatically trigger when profit drops below 0 to quickly highlight loss-making products and support better decisions.

Key Features

Top & Bottom product performance

Sales & Profit tracking

KPI cards for business overview

Interactive visuals & filters

Automated insight alert: “Negative Profit”

Tools Used

Power BI Desktop

Power Query

DAX
